## 排序演算法

### 1. 氣泡排序法 (Bubble Sort)
 氣泡排序法是一種最基礎的排序方法，透過相鄰兩個元素互相比較，若順序錯誤就交換位置。
每一輪比較後，較大或較小的數會像氣泡一樣慢慢移到正確位置。
優點是概念簡單、容易實作；缺點是當資料量大時，執行效率較差。介紹氣泡排序的原理和實現方式 

### 2. 選擇排序法 (Selection Sort)
介紹選擇排序選擇排序法的作法是，從尚未排序的資料中，每次選出最小值（或最大值），再放到已排序區域的正確位置。
它的交換次數通常較少，邏輯也很清楚。
但是整體比較次數仍然偏多，因此效率不算高，適合用來學習排序原理。的原理和實現方式 

### 3. 插入排序法 (Insertion Sort)
介紹插入排序的原理插入排序法是將資料一筆一筆插入到已排序好的序列中，直到全部資料都排好。
它的概念很像打撲克牌時整理手牌的方式。
當資料量不大，或原本資料已經接近排序完成時，插入排序法的效率會不錯。和實現方式

### 4. 合併排序法 (Merge Sort)
介紹合併排序合併排序法採用分治法的概念，先將資料不斷切成兩半，直到每組只剩下一個元素，再逐步合併成有序序列。
這種方法的效率比前面幾種基本排序法更好，尤其適合處理大量資料。
不過它通常需要額外的記憶體空間來協助合併。的原理和實現方式

### 5. 快速排序法
 其他排序演算快速排序法也是一種常見的分治法排序。
它會先選定一個基準值（pivot），再將資料分成比基準值小和比基準值大的兩部分，接著對左右兩邊重複進行相同動作。
快速排序法平均效率很高，是實務上很常見的排序方式，但若基準值選得不好，效能可能下降。

### 綜合比較
比較各種排序演算各種排序法都有不同特性。
氣泡排序、選擇排序、插入排序結構簡單，適合初學者理解排序概念；

若重視程式易懂與教學用途，可選擇基本排序法；若重視執行效率，則通常會考慮合併排序或快速排序。法的時間複雜度、空間複雜度和實際效能
